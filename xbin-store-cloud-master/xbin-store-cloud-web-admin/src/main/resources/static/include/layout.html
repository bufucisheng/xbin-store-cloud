AdminLTe有3中布局风格，根据需求现在增加一种，分别如下

固定（fixed）

在body节点添加 fixed 类名

固定页头和侧边栏在浏览器可视区域。

demo

隐藏侧最小化（sidebar-collapse）

在body节点添加类 sidebar-collapse sidebar-mini

默认图标显示侧边栏，可通过页头的菜单按钮切换，如果不加sidebar-mini类则完全隐藏侧边栏

demo

头部导航（layout-top-nav）

在body节点添加 layout-top-nav 类名

只有头部导航，没有侧边栏。

demo

盒子布局（layout-boxed）

在body节点添加 layout-boxed 类名

页面最大宽度为1250px，不再满屏适应页面

demo

非跳转模式（ajax-template）（新增）

在body节点添加 ajax-template 类名

页面不再跳转，而是通过ajax加载的形式加载页面

demo
